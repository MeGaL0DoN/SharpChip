# SharpChip

Requires: NET 8

Keyboard Layout:
| 1 | 2 | 3 | 4 |
| --- | --- | --- | --- |
| Q | W | E | R |
| A | S | D | F |
| Z | X | C | V |

Press TAB to pause, ESC to reload current ROM.
